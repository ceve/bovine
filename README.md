# Cattle Quiz

Simple Node.js + Express quiz where you guess cattle breeds (Hereford, Angus, Charolais, Limousin, Simmental, Highland).

## Run locally

```bash
npm install
npm start
```

Then open: `http://localhost:3000`

## Dev mode

```bash
npm run dev
```

## Environment

- `PORT` (optional): defaults to `3000`
